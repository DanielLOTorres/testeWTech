## Inicialização 

Para iniciar o projeto será necessário rodar o comando:

npm install 

Tanto da pasta api, quanto na pasta web


## Execução 


Após instalação, rodar 

npm start 

Para iniciar a api, e 

npm run serve 

Para iniciar o projeto front-end
